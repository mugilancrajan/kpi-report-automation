Problem: "ExecutionPolicy" error
→ ☐ Run PowerShell as Administrator
→ ☐ Try the command again

Problem: "Permission denied"
→ ☐ Use Personal Access Token instead of default password
→ ☐ Follow Step 5, Option B

Problem: Script stops with an error
→ ☐ Take a screenshot of the error
→ ☐ Try again
→ ☐ If it persists, run manually: git push -u origin main

Problem: Can't find the script
→ ☐ Make sure you're in correct folder
→ ☐ Check that PUSH_TO_GITHUB.ps1 exists there
